# Portfólio Pessoal — Ari da Silva Junior

Este repositório contém o código fonte do meu portfólio pessoal para a atividade prática da disciplina "Ferramentas de Desenvolvimento Web".

## Arquivos
- index.html — Página única com 4 seções: Sobre, Formação, Portfólio, Contato.
- styles.css — Estilos CSS3 sem frameworks.
- script.js — JavaScript para validação, simulação de envio, menu responsivo e tema claro/escuro.

## Como executar localmente
1. Clone este repositório.
2. Abra `index.html` no navegador (duplo clique) ou sirva via Live Server no VS Code.

## Deploy (GitHub Pages)
1. Crie um repositório público no GitHub chamado `portfolio-Ari_Junior` (ou similar).
2. Faça commit dos arquivos (`index.html`, `styles.css`, `script.js`) na branch `main`.
3. No GitHub vá em *Settings > Pages* e selecione a branch `main` (root) como source. Salve.
4. O site ficará disponível em `https://github.com/Ari0Junior/Trabalho-Uninter-Desenvovimento-web`.
5. No PDF de entrega inclua:
   - Link do site publicado
   - Link do repositório público no GitHub
   - Código-fonte completo (copiado/colado nos campos exigidos)
   - Prints de cada seção com a barra de endereço mostrando o link (evitar `localhost`)

## Observações
- Não usei frameworks (Bootstrap/Tailwind/jQuery/React etc).
- Código comentado onde relevante.
